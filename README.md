# Calculator Project

A simple calculator project written in Python for learning Git and GitHub. This project includes a calculator application and a number guessing game.

## Features

- **Calculator Operations**: Addition, subtraction, multiplication, division, power, and modulus
- **Number Guessing Game**: Computer tries to guess your number
- **Error Handling**: Division by zero protection and invalid input validation
- **Modular Structure**: Separate files for core logic and user interface

## Project Files

```
calculator-project/
├── main.py           # Calculator user interface
├── calculator.py     # Calculator mathematical functions
├── guess_number.py   # Number guessing game
├── utils.py          # Helper functions
└── README.md         # Project documentation
```

## Installation

1. Clone the repository:
   git clone https://github.com/mahdyeh346/calculator-project.git
Navigate to the project folder:


cd calculator-project
Run the calculator:


python main.py
Run the guessing game:


python guess_number.py
How to Use
Calculator:
After running main.py, you'll see a menu with these options:

1: Addition (+)

2: Subtraction (-)

3: Multiplication (*)

4: Division (/)

5: Power (**)

6: Modulus (%)

7: Exit

Enter the operation number and then provide two numbers.

Number Guessing Game:
Run guess_number.py. The computer makes a guess, and you guide it by entering:

0: Your number is smaller

1: Your number is bigger

2: Equal (correct guess)

3: Exit game
